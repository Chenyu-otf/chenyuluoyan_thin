# 【辰宇落雁體】── 複寫呢喃，為你成篇
![chenyu2 0 _github](https://github.com/user-attachments/assets/73225485-1b7b-42c5-8a01-f7414fc1f55e)

「辰宇落雁體」是我與韋辰在高中的自主學習計畫，由韋辰負責書寫，我進行後製而成。手寫字型就像是我們將字詞複寫於紙上，讓字跡留下，以這種近似雋永的形式封裝，使被複寫的日常沒有保存期限，也像是一種永恆的召喚，在任何無法抵達的以後，都能為你書寫成篇。

## 製作故事
### 欸，你要不要做手寫字體？
去年的八月，那陣子我開始關注字型相關議題，在一次偶然看到韋辰的手寫字後，萌生出要是自己也能做一套手寫字體，真是太酷了！而適逢 108 課綱的我們，課綱強調自主學習的精神，正巧讓我們有機會發展計畫，於是，手寫字體的鑽研之路就這樣開始了⋯⋯

但「萬事起頭難」，我們一開始的起步並不這麼順利，對於字型不太熟悉的我們於是開始搜尋相關資訊，忽而地想到，YouTube 頻道「壹加壹」與 [justfont](https://justfont.com/ "justfont") 推出的粒線體，我們寫信請教手寫字型的前期製作方向，獲得了他們真誠的回覆，他們向我們分享製作字型需注意的要點，也在計畫過程中提供我們諮詢與技術指導，成為很大的助力。

我們最初是在線上字型製作平台──「[守寫字網站](https://www.writes.com.tw/ "守寫字網站")」習得手寫字型的製作流程，後來為了更大程度的製作自由，我們嘗試拆解流程，並以自己的方式進行製作，然摸索的過程並非一蹴可及，在那段期間的每週四，我潛心鑽研的同時，韋辰就在一旁書寫，兩個人窩在圖書館的角落為心中的理想並肩努力著，製作字體是枯燥的，日復一日的書寫、修改與後製，現在想想，我們還挺浪漫的，為了完成理想，不惜代價的繞了點路，也有好幾次洩氣的想著，或許真的做不出來了，是對於成果的想像才讓我們堅持下來。

在字體做出成果後，也陸續獲得不少稱讚，神奇的是，製作過程中並未覺得特別的我們，在第一次打出完整的句子時，卻被感動了。若浪漫一點來說，製作手寫字體這件事，似將字詞複寫於紙上，將字跡留下為人使用。

回想製作的契機，那句「欸，你要不要做手寫字體？」在今天終於實現了，韋辰寫下這些文字後也將與十六歲告別，正所謂字如其人，期望她未來回首，能在字裡行間看見自己。

### 特別感謝
在計畫初期，我們對於製作毫無概念的時候，第一個想到可以請教的對象就是 justfont，當時只覺得，如果能得到大略的指點就很感謝了，所以在收到他們熱心的回覆與幫助的時候其實很受寵若驚，專業團隊的幫助真的成為我們很大的助力，在這裡特別感謝曾國榕設計師與 justfont 團隊於本計畫提供諮詢、技術指導與協助調整。不善言詞，但我們真的萬分感謝。

### 寫在2025/02/24
1.0 推出的 900 多天裡，辰宇落雁體代替我們去到了很多地方，謝謝喜歡字體的大家，一個自主學習計畫帶我們完成了很多沒想過的事情，很榮幸辰宇落雁體成為臺灣字型當中的一小片風景，再次感謝一路支持的你們與當初贊助支持的人們，謝謝你們。

最後，我們想特別感謝國榕、justfont 一直以來的支持與協助。三年多前，在圖書館的三樓、小小的座位與兩位懵懂的高中生，這一切怎麼開始的，我們不會忘記。

- 自由使用是否可以商用呢？
<br>可以喔！歡迎大家下載使用辰宇落雁體，開源授權下不會被收取任何費用，可以將字體運用在各個地方，不論是印刷、各種圖文設計等各種商用、自用都很歡迎！

<br>開源授權下，可以上傳字體到任何裝置，如果是iOS系統上的話，推薦使用「下筆」APP，將文字複製轉貼至限時動態！（詳情請搜尋「下筆」）
