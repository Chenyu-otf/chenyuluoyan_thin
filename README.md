# 【辰宇落雁體】── 複寫呢喃，為你成篇
「辰宇落雁體」取自兩位作者名字，由劉韋辰用原子筆於稿紙上書寫，王立宇進行後期製作而成的手寫字型，以OFL 1.1開源授權釋出，歡迎自由使用、修改與散布（註：「辰宇落雁」為此字型專案的保留名稱，改作字型不得使用其作為字型名稱）
